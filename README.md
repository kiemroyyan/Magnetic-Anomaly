Anomali magnetik memberikan informasi penting tentang variasi sifat magnetik batuan di bawah permukaan bumi.  
Dengan mengolah data medan magnet total, kita dapat memetakan struktur bawah permukaan seperti sesar, intrusi batuan, atau batas formasi geologi.

Proyek ini dikembangkan menggunakan Python karena kemampuannya yang kuat dalam analisis data, visualisasi, dan integrasi dengan pustaka geospasial.
